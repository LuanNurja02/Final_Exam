# Numerical Methods Exam

This repository contains the work for the **Numerical Methods exam**.

## Description

The exam is divided into two main exercises:

1. **Document Search**  
   The first exercise focuses on document retrieval, including the use of **terms** and **queries** to search and analyze documents.

2. **Regression Exercise**  
   The second exercise consists of a **regression problem**, where numerical methods are applied to model and analyze data.
